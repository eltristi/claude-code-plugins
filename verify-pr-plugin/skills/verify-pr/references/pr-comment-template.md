# PR Comment Template

Use this template when posting verification results to the PR.

## Successful Verification (with videos)

```markdown
## PR Verification — Visual Test Results

**PR:** #<NUMBER> — <TITLE>
**Platforms tested:** <PLATFORMS>
**Branch:** `<BRANCH>` at `<SHORT_SHA>`

### Test Scenarios

#### 1. <scenario-name>
- **Status:** :white_check_mark: Passed
- **Video:** <GYAZO_URL>
- **What was tested:** <BRIEF_DESCRIPTION>

#### 2. <scenario-name>
- **Status:** :white_check_mark: Passed
- **Video:** <GYAZO_URL>
- **What was tested:** <BRIEF_DESCRIPTION>

---

**Summary:** <PASSED>/<TOTAL> scenarios passed

<sub>Generated by verify-pr</sub>
```

## Template Variables

| Variable | Source |
|----------|--------|
| `<NUMBER>` | PR number from `gh pr view` |
| `<TITLE>` | PR title |
| `<PLATFORMS>` | e.g., "iOS Simulator, Android Emulator" or "Web (Chrome)" |
| `<BRANCH>` | Branch name |
| `<SHORT_SHA>` | First 7 chars of HEAD commit SHA |
| `<GYAZO_URL>` | URL returned from Gyazo MCP upload |
| `<PASSED>` / `<TOTAL>` / `<FAILED>` | Scenario counts |
| `<ERROR_MESSAGE>` | Captured from Maestro/Playwright output |
